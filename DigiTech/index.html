2. INIT: Final Project Goal

                    

                        Welcome to your final project! The goal is to use your HTML/CSS skills in CodePen to build a 3-page website that presents all the work you've done in this 9-week course. Your "Digital Portfolio" will showcase your Python code, data analysis, and AI research, along with your reflections on digital citizenship.
                    

                


                

                    
The Digital Portfolio Structure 📱

                    
Your portfolio website should include the following sections:

                    
    
                        
  • Homepage Section: A main section that introduces you and has links to your other sections.
    • 
                        
  • "My Code" Section: Your final Python code with a write-up explaining what it does.
    • 
                        
  • "My Research" Section: Your data project and AI ethics report (with Digital Citizenship reflection).
    • 
                        
  • Published & Live: The final site must be viewable via its public CodePen link.

                                    Digital Content (HTML/CSS) (40%)
                                    The CodePen exists but has broken links, missing content, and little to no CSS styling.
                                    The CodePen meets all requirements. It has all required sections, links work, and all project work is embedded. CSS is used and functional.
                                    The CodePen is polished, responsive, and easy to navigate. The design and layout show creativity and planning.
                                    The design justifies choices based on UX principles (e.g., accessibility, contrast). The student uses advanced features (e.g., CSS animations) to enhance communication, not just decoration.
                                

                                

                                    Programming & CT (Python) (20%)
                                    The Python project is included but is incomplete or has significant errors.
                                    The Python project is complete, runs correctly, and meets the Y7/Y8 requirements (e.g., Y8 has loops).
                                    The Python code is clean, well-commented, and includes extension features beyond the basic requirements.
                                    The code generalises the solution. It uses efficient data structures (e.g., lists/dictionaries) instead of repetition and handles unexpected user input gracefully.
                                

                                 

                                    Data & AI Research (30%)
                                    The Data and AI projects are missing or are very brief and show little understanding.
                                    The Data and AI reports are complete and meet the Y7/Y8 requirements (e.g., Y8 has a chart and discusses ethics).
                                    The reports show deep insight. The data is well-cleaned and the AI ethics discussion is well-reasoned.
                                    The student critically evaluates the limitations of the dataset or AI tool itself. They propose specific, actionable solutions to the ethical risks or data bias identified.
                                

                                

                                    Digital Citizenship (Reflection) (10%)
                                    The reflection is missing or is a single sentence.
                                    The student includes a reflection that explains what they learned about digital citizenship.
                                    The reflection connects concepts, explaining how digital citizenship relates to AI bias or data privacy.
                                    The reflection advocates for a change in behaviour. It generalises the specific lesson to broader principles (e.g., "Digital Rights") that apply to future technologies.
